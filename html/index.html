This project uses the LAMP software bundle, with Python 3 in the back end (implemented as the CGI handler for the Apache server). No other frameworks are used. For now, localhost is used as both the development and production server.

The website, its design and associated logic is the result of hard work by the students Irfan, Kanti & Tarik. It is developed as the interface for their final Database Management project for this course.
